# CELO Inventory Management System

## Introduction
**CELO Inventory Management System** is a centralized asset tracking solution designed to streamline inventory operations and hardware lifecycle management within BPO environments.
This system provides real-time visibility into company assets, enabling administrators to efficiently monitor hardware assignments, inventory availability, maintenance records, and asset movement across departments and facilities.

With features such as personnel asset assignment, maintenance logging, audit-ready reporting, and inventory analytics, the platform minimizes asset loss, improves accountability, and enhances operational efficiency.
Tailored for the high-volume operations **CELO Business Solutions Inc.**, this system ensures accurate tracking of organizational assets from procurement to retirement.

---

## Features
- Executive Dashboard with real-time inventory statistics
- Asset Inventory Master List
- Add, Update, Remove, and View Assets
- Personnel Asset Assignment Workflow
- Asset Return and Reassignment Tracking
- Maintenance and Repair Logging
- Asset History and Audit Trail
- Multi-status Asset Monitoring
- CSV Export for Reports
- Search and Multi-column Filtering
- User Role Management
- Inventory Reports and Analytics

---

## Technologies Used
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Libraries/Plugins:** DataTables, Chart.js, Axios
